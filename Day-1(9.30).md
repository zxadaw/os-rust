### 在电脑上构建了基本的开发环境，了解了在做exercise过程中必须的一些命令行指令，包括git指令和rust指令

```rust
呈现完成情况：cargo run list
接下里的一个错误位置：cargo run watch
编译并运行：cargo build/run
(也可以用rustlings）

git提交：git commit -m "说明文字"
提交到github： git push
给出需要提交的文件： git add *

```
